    1.安装及用法

    安装

    安装前你需要最新稳定版 Node.js, 和 npm (Node Package Manager). 借助 npm 可以安装 CoffeeScript:

    sudo npm install -g coffee-script

    如果不需要全局安装，则可以去掉-g

    3.优点

    • 更少，更紧凑，和更清晰的代码
    • 通过规避和改变对JavaScript中不良部分的使用，只留下精华，让代码减少出错率，更容易维护
    • 在很多常用模式的实现上采用了JavaScript中的最佳实践
    • CoffeeScript生成的JavaScript代码都可以完全通过JSLint的检测

      4.总结

      关于CoffeeScript的介绍就到此为止。CoffeeScript能极大简化JavaScript的书写，对JavaScript中的精华与糟粕做了有用的取舍。但是由于他对js的精简，封装了很多细节，方便应用的同时也让我们对原理的了解没有那么深入，所以不应该在不理解js的情况下直接用coffeeScript，它只是我们一个更加高效书写js的工具。
